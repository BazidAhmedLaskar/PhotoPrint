# Background Remover Feature - Setup & Tutorial

## What is the Background Remover?

The Background Remover is a powerful feature in PhotoPrint Pro that automatically removes the background from your passport photos and replaces it with a solid color. This is perfect for:
- Creating professional ID photos with clean backgrounds
- Removing shadows and cluttered backgrounds
- Meeting passport photo requirements for multiple countries
- Quickly processing multiple photos at once

---

## How It Works

1. **AI-Powered**: Uses the remove.bg API to intelligently detect and remove backgrounds
2. **One-Click Processing**: Simply click "Remove Background" to process all images at once
3. **Customizable Colors**: Choose from preset colors (white, light blue, grey) or set a custom color
4. **Reversible**: Can easily restore original backgrounds if needed

---

## Getting Started: Setup in 3 Steps

### Step 1: Get a Free remove.bg Account

1. Visit [https://www.remove.bg/api](https://www.remove.bg/api)
2. Click "Sign Up" and create a free account
3. Verify your email

**Free Plan Includes:**
- 50 API calls per month (50 background removals)
- High-quality transparent PNG output
- Perfect for personal use and small batches

### Step 2: Copy Your API Key

1. Log in to your remove.bg account
2. Go to the "API" section in your dashboard
3. Copy your **API Key** (usually starts with a long alphanumeric string)
4. Keep this key safe - treat it like a password

> **Note:** Your API key is unique to your account. Never share it publicly.

### Step 3: Add Your API Key to PhotoPrint Pro

1. Click the **⚙️ API Setup** button in the top navigation bar
2. Paste your API key into the input field
3. Click **🧪 Test Key** to verify it works (you'll see a success message if it's valid)
4. Click **✓ Save & Continue**

Your API key is now saved to your browser's local storage and you're ready to use background removal!

---

## Using Background Remover

### Basic Steps:

1. **Add Images** using the "➕ Add Image" button
2. **Click "✂️ Remove Background"** in Step 3
3. Wait for processing (a few seconds per image)
4. **See the magic** - your photo now has a clean background!
5. **Choose a Background Color** - White, Light Blue, Grey, or Custom
6. **Print or Download** your processed images

### With Batch Processing:

- Remove backgrounds from multiple images at once
- All images are processed using a single API call each
- Track progress with real-time status updates

## Troubleshooting

### Problem: "Invalid API key"
**Solution:** 
- Go back to remove.bg and copy your API key again
- Make sure you copied the entire key without extra spaces
- Check that your account is confirmed/verified at remove.bg

### Problem: "Quota exceeded"
**Solution:**
- Your 50 monthly API calls are used up
- Upgrade your remove.bg plan (paid plans start at $99/month for 500 calls)
- Or wait until next month for your quota to reset

### Problem: "Network error"
**Solution:**
- Check your internet connection
- Try again in a moment (temporary server issues)
- refresh the page if the error persists

### Problem: "API key is too short"
**Solution:**
- remove.bg API keys are always 20+ characters long
- Copy your key directly from remove.bg without any modifications
- Look for your "API Key" in the API dashboard, not other tokens

### Problem: Images not being removed
**Solution:**
- The API key might be invalid - test it using "🧪 Test Key"
- You might be out of free monthly quota (50 calls)
- Try with a different image to isolate the issue

## Frequently Asked Questions (FAQ)

### Q: Is my photo data safe?
**A:** Your photos are sent directly to remove.bg's servers for processing. They delete images from their servers after processing. See their privacy policy at remove.bg for details.

### Q: Can I use remove.bg elsewhere?
**A:** Yes! Your remove.bg account works everywhere. Your API key can be used in other applications too.

### Q: What if background removal doesn't work perfectly?
**A:** Background removal quality depends on:
- Image quality and lighting
- How distinct the subject is from the background
- Hair or transparent objects (these are harder to detect)
Try adjusting the image (better crop, brightness) and retry.

### Q: Can I get more than 50 calls per month?
**A:** Yes! remove.bg offers paid plans:
- Starter: $99/month (500 calls)
- Professional: $199/month (1,500 calls)
- Plus higher plans available

### Q: Will my API key work on mobile?
**A:** Yes! PhotoPrint Pro works on mobile devices, and your API key will work the same way.

### Q: What if I lose my API key?
**A:** You can regenerate it anytime in your remove.bg account settings.
