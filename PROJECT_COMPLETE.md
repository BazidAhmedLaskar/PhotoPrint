## 🎯 USAGE & FEATURES

### Basic Workflow
1. **Upload**: Click or drag image
2. **Adjust**: Move sliders for real-time preview
3. **Apply**: Select filter or transform
4. **Refine**: Layer multiple adjustments
5. **Compare**: Toggle before/after
6. **Download**: Export as JPG or PNG

### Passport Photo Creation
```
1. Load your photo
2. Crop to face
3. Click "📷 Passport Photo"
4. Select layout (4×6, 6×8, or 8×12)
5. Download PNG
6. Print A4 ready-to-cut photos
```

### Image Compression
```
1. Load image
2. Click "📦 Compress"
3. Enter target size (e.g., 200 KB)
4. Download optimized image
5. Perfect for email/forms
```

### Document Form Photos
```
1. Load image
2. Click "📋 Resize 4 Forms"
3. Select size: 50KB, 100KB, 200KB, 500KB
4. Download optimized
5. Ready to upload
```
